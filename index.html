Tic-Tac-Toe
Tic-Tac-Toe

This project represents my first foray into web development. I developed a single page application that allows an authenticated user to play Tic-Tac-Toe. The game logic was created with JavaScript, and the game board was created with Bootstrap. jQuery was used to communicate with an API that was provided to us. Some of the features I would like to add in the future are a method for recordkeeping and multi-player functionality.

Fantasy Baseball
Fantasy Baseball

My first attempt at developing a full stack web application. An authenticated user can log in and CRUD fantasy leagues. The back end API was created using Ruby on Rails, with JavaScript, jQuery, Bootstrap and Handlebars for the client. Some of the features I would like to add in the future are the ability to create teams and players with seeded data, with acculumulated category totals displayed.

Survey Says!
Survey Says!

For this project, I collaborated with two other web developers to develop a single page application that allows an authenticated user to create surveys. They are also able to view all surveys created by other users and the results. The back end API was created using Express, with JavaScript, jQuery and Handlebars for the client. Git, GitHub and pair programming were used for version control. For future versions, we would like to limit users to only answer a survey one time, and to perhaps use a chart for results.

Pokemon Go Raid Leaderboard
Team Instinct Boston Raid Leaderboard!

I consulted a local stakeholder in the Pokemon Go community about creating an app that would make their lives easier. Using React and Ruby on Rails, I developed an app that allowed an authenticated user to create a raid entry and view all raid entries created by other users. I would love to continue working on this app and adding a means for users to sort raids by time, view raids per boss, and even display their own photos to confirm their time.
